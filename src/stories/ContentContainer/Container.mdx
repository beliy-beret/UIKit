# ContentContainer

Контайнер ограничивающий ширину контента, представляет из себя html тег div с задаными отступами для разных экранов.\
Принимает параметры свойственные html div тегу.\
Максимальная ширина 1366px, по умолчанию позиционируется по центру страницы.
